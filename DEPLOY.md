# Deploy FaceAttend

FaceAttend has three parts:

| Part | Host | Why |
|------|------|-----|
| Frontend (React) | **Netlify** | Static SPA + API proxy |
| Backend (Node API) | **Render** | Database, auth, file uploads |
| AI service (Python) | **Render** | Face recognition (Docker) |

Netlify cannot run the Node API or Python AI service — only the frontend.

---

## Step 1 — Deploy API + AI on Render

1. Go to [Render Dashboard](https://dashboard.render.com) → **New** → **Blueprint**
2. Connect GitHub repo: `palindrome-kasak/attendence`
3. Render reads `render.yaml` and creates two services:
   - `faceattend-ai` — Python face recognition
   - `faceattend-api` — Node.js API
4. Wait for both to finish building (AI Docker build takes ~10–15 min first time)
5. Copy the **faceattend-api** URL, e.g. `https://faceattend-api.onrender.com`

> **Note:** Render free tier spins down after inactivity. First request may take ~30s to wake up.

Test the API:

```bash
curl https://faceattend-api.onrender.com/api/health
```

---

## Step 2 — Deploy frontend on Netlify

### Option A — Connect GitHub (recommended)

1. Go to [Netlify](https://app.netlify.com) → **Add new site** → **Import an existing project**
2. Choose GitHub → repo `palindrome-kasak/attendence`
3. Build settings (auto-detected from `netlify.toml`):
   - Base directory: `frontend`
   - Build command: `npm ci && npm run build:netlify`
   - Publish directory: `frontend/dist`
4. Add environment variable:
   - `BACKEND_URL` = `https://faceattend-api.onrender.com` (your Render API URL, no trailing slash)
5. Deploy

### Option B — Netlify CLI

```bash
cd frontend
npm ci
BACKEND_URL=https://faceattend-api.onrender.com npm run build:netlify
npx netlify-cli deploy --prod --dir=dist
```

## Environment variables

### Netlify

| Variable | Example |
|----------|---------|
| `BACKEND_URL` | `https://faceattend-api.onrender.com` |

### Render — faceattend-api

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | `file:./dev.db` (set in render.yaml) |
| `JWT_SECRET` | Auto-generated by Render |
| `AI_SERVICE_URL` | `https://faceattend-ai.onrender.com` |
| `UPLOAD_DIR` | `uploads` |

### Render — faceattend-ai

| Variable | Description |
|----------|-------------|
| `PORT` | Set automatically by Render |

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Login fails on Netlify | Check `BACKEND_URL` is set and API health returns OK |
| API timeout on first load | Render free tier waking up — wait 30s and retry |
| Face scan fails | Ensure `faceattend-ai` service is running; check Render logs |
| Camera not working | HTTPS required — Netlify provides this automatically |
